# 快速了解 HabitPulse

HabitPulse 是一款专为 Android 设备设计的习惯追踪应用，致力于帮助用户建立和维持良好的日常习惯。通过简洁直观的界面设计和智能化的提醒机制，让习惯养成变得更加轻松有效。

## 核心特性

### 习惯管理
- **灵活创建**：支持自定义习惯标题、备注和重复周期（每日/每周）
- **智能提醒**：可设置多个提醒时间，不错过任何一次打卡
- **拖拽排序**：长按进入多选模式，自由调整习惯顺序
- **批量操作**：支持批量删除，快速清理不再需要的习惯

### 打卡记录
- **完整历史**：记录每次打卡的时间戳、本地日期和时区信息
- **日期筛选**：通过日期选择器快速查看特定时间段的完成情况
- **习惯过滤**：可按单个习惯或全部习惯筛选记录
- **可视化统计**：清晰展示完成进度和趋势

### 社交监督
- **联系人监督**：为习惯添加监督人（邮箱/电话），让他人帮助你坚持
- **责任提醒**：习惯完成情况可通知指定联系人，增强坚持动力

### 精美设计
- **Material Design 3**：采用最新的 MD3 设计规范
- **动态配色**：支持 Android 12+ 的 Material You 主题色
- **响应式布局**：自适应手机和平板，横竖屏完美适配
- **流畅动画**：自定义屏幕过渡和卡片动画效果

## 技术架构

HabitPulse 采用现代化 Android 开发技术栈：

| 组件 | 说明 |
|------|------|
| **Jetpack Compose** | 声明式 UI 框架，现代化开发体验 |
| **Material Design 3** | 采用最新的 Material Design 设计规范 |
| **Room 数据库** | 本地数据持久化，离线可用 |
| **ViewModel + Flow** | 响应式架构，数据驱动 UI |
| **Navigation Compose** | 导航组件，实现流畅的页面切换动画 |
| **DataStore** | 现代化的偏好设置存储方案 |
