# 空间清理

> [!WARNING] 高级设置
> 本页设置内容属于高级设置，确保你已经了解各操作和设置再操作。拿不准的话，找一个懂行朋友陪同操作。

> [!IMPORTANT] 重要提示
> 本站教程仅适用于最新版本的 HabitPulse，如果你设备上的 HabitPulse 与教程里不一样，请更新 HabitPulse。

HabitPulse 在应用内浏览器使用缓存保证日常体验迅速，缓存会随时间累积。

你可以在使用一段时间后清理缓存。清除这部分缓存不影响你的习惯打卡数据。

清理缓存后，你可能会觉察到下次访问网页时需要稍微更长的时间加载。

## 清理缓存

在设置中，轻点「清理缓存」并在确认对话框中轻点「清理」，会清除掉应用内浏览器的缓存，但保留你在应用内浏览器里的登录信息。

大多数情况下，清理缓存就已足够。

## 深度清理

在设置中，轻点「深度清理」并在确认对话框中轻点「清理」，会清理掉应用内浏览器的缓存和你在应用内浏览器里的登录信息。

一般情况下，深度清理和清理缓存清理的空间非常接近。除非你希望在不清除习惯和打卡数据的情况下，完全清除应用内浏览器的数据，否则不应该优先考虑使用深度清理。

## 需不需要频繁清理缓存？

请**不要**频繁清除缓存。因为：
- 缓存可以用于加速网页访问，部分访问过的页面（例如本站的页面）可以之后离线访问。
- 缓存不会占据太多空间。HabitPulse 应用内浏览器只访问 HabitPulse Docs 和 HabitPulse GitHub 存储库的情况下，长期产生的缓存小于 100 MB。

如果存储空间告罄，一般不是 HabitPulse 造成的，频繁清理 HabitPulse 也不会有显著帮助。
