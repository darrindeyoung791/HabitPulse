# 保持 HabitPulse 后台运行

> [!WARNING] 高级设置
> 本页设置内容属于高级设置，确保你已经了解各操作和设置再操作。拿不准的话，找一个懂行朋友陪同操作。

> [!IMPORTANT] 重要提示
> 本站教程仅适用于最新版本的 HabitPulse，如果你设备上的 HabitPulse 与教程里不一样，请更新 HabitPulse。

HabitPulse 需要持续在后台运行，才能够及时通知你完成习惯。

## 为 HabitPulse 后台卡片加锁

打开 HabitPulse，进入任务切换界面，给 HabitPulse 应用卡片加锁即可。此后不要手动划掉 HabitPulse 后台卡片。

对于多数系统，可以通过向下滑动应用卡片并释放，或长按卡片轻点菜单里的上锁，或点开菜单轻点上锁来锁定后台。

## 打开常驻通知

打开 HabitPulse 设置，轻点打开「保持后台运行」开关。此时 HabitPulse 将发送持久通知，保证后台运行。

要打开此功能，需要授予 HabitPulse 通知权限。

## 修改省电与启动策略

去 HabitPulse 应用信息里，找到电量，打开后台耗电管理，选择「允许后台运行」（部分手机可能是「允许后台高耗电」）。

去手机管家，在电池管理中找到 HabitPulse，关闭「自动管理」，打开「允许自启动」「允许关联启动」「允许后台启动」。

> [!TIP] 提示
> 上述操作在不同的系统中可能略有不同。

## 常见问题

::: details HabitPulse 耗电多吗？

HabitPulse 在后台耗电量非常低，因为 HabitPulse 十分小巧，没有多余功能在后台运行。

典型情况下，HabitPulse 每周耗电约 400mAh。
:::
