# 强制使用平板横屏模式

> [!WARNING] 高级设置
> 本页设置内容属于高级设置，确保你已经了解各操作和设置再操作。拿不准的话，找一个懂行朋友陪同操作。

> [!IMPORTANT] 重要提示
> 本站教程仅适用于最新版本的 HabitPulse，如果你设备上的 HabitPulse 与教程里不一样，请更新 HabitPulse。

HabitPulse 为各种屏幕尺寸的设备精心适配布局。同样的页面，在手机平板上可能显示不一样的布局。

如果你使用小平板、折叠屏手机或显示大小特殊的设备，HabitPulse 可能错误地显示手机界面。你可以手动打开平板模式，提升操作效率。

## 开启平板模式

在设置中，轻点打开「强制使用平板横屏模式」开关并在确认对话框中轻点「确定」，会强制在当前设备进入平板模式。

如果 HabitPulse 认为你使用的设备是平板，则不显示此开关。

## 开启后的行为

开启强制使用平板横屏模式后：

| 横屏下的功能模块 | 平板模式 | 手机模式 |
|---------|----------------|---------------------|
| 主页导航侧栏 | 默认展开，可通过按钮手动收起/展开 | 不可展开 |
| 主页顶部应用栏 | 页面在顶部时展开；收起时仍占据一行 | 页面在顶部时显示一行收起内容；滚动时完全收起 |
| 记录界面（日期选择按钮） | 默认展示为日历选择模式 | 默认展示为手动输入模式，以空出足够的垂直空间 |

## 已知问题

如果你使用手机，或横屏时左右侧屏幕有前置摄像头时，当摄像头处于主页导航侧栏一侧时，收起主页导航侧栏时图标会受挤压变小。
