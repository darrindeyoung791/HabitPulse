## 关于 HabitPulse

HabitPulse 是 [HabitPulse 开发组](/team.md) 精心制作的习惯保持应用。因具有按时提醒他人的功能，故使用「脉冲信号（Pulse）」作为名称的一部分。

> [!WARNING] 重要提示
> 「HabitPulse」、带有 H、P 两个字母的图标等元素并不代表或暗示与任何现有的名称、图标、公司或商标等相关。
> 
> 如有侵权，请 [联系我们修改](https://github.com/darrindeyoung791/HabitPulse/issues)。
> 
> 如果你有更好的名称或图标，欢迎 [留下你的想法](https://github.com/darrindeyoung791/HabitPulse/issues)。

HabitPulse 是开源软件。你可以**免费**下载使用，也可以在**遵守许可证**的情况下，在 HabitPulse 的基础上进行二次开发。源码见 [GitHub](https://github.com/darrindeyoung791/HabitPulse)。

要查看 HabitPulse 使用的开源项目及许可证，请安装 HabitPulse，打开应用，进入设置，轻点「开放源代码许可」。
