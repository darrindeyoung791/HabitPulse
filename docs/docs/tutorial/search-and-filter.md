# 搜索与筛选

> [!IMPORTANT] 重要提示
> 本站教程仅适用于最新版本的 HabitPulse，如果你设备上的 HabitPulse 与教程里不一样，请更新 HabitPulse。

## 搜索习惯、联系人

处于习惯、联系人页面时，轻点右上角的放大镜图标，会弹出搜索框。

在搜索框输入文本即可搜索。

## 筛选记录

处于记录界面时，你可以按日期和习惯筛选打卡记录。

### 查看某天的打卡记录

轻点右上角的日历图标，会弹出一个日期选择器。选中某一天，即可只查看当天的全部记录。

选中日期时，顶部图标由原先的日历图标变为选中的日期和「❌」组成的按钮。点击这个按钮取消选中日期，显示全部记录。

> [!TIP] 日期选择器怎么用？
> 在日历部分左右滑动，或点击左右箭头，可以切换月份。
>
> 点击标有「xxxx 年 xx 月」的下拉菜单，可以快速切换年份。
>
> 点击顶端的笔图标，可以切换到键盘输入。
>
> 当处于手机横屏状态时，日期选择器会自动变为键盘输入，避免垂直空间不足。

### 查看某习惯的打卡记录

轻点左上角标题下方的打勾下拉菜单，从列表里选中一项习惯，即可只查看这个习惯的全部记录。
