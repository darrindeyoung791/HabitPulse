# 初次上手 HabitPulse

HabitPulse 非常易于上手。即使是初次使用，也能飞快适应。

可以按顺序看完每篇文章，亦可善用本页面的搜索框找到需要的信息。

> [!IMPORTANT] 重要提示
> 本站教程仅针对最新版本的 HabitPulse，如果你设备上的 HabitPulse 与教程里不一样，请更新 HabitPulse。

## 了解各个界面

### 1. 习惯界面

习惯界面是应用启动时的默认界面，你添加的习惯以卡片列表样式显示在习惯界面。

通过轻点屏幕底部或侧边的「习惯」导航按钮，可以进入习惯界面。

处于习惯界面时，轻点屏幕底部或侧边的「习惯」导航按钮可以回到顶部。

#### 1.1 习惯创建界面

这个界面同时用于新建习惯和编辑现有习惯。

通过轻点习惯界面右下角的「新建习惯」按钮，可以进入习惯创建界面。

通过长按任一现有习惯，再轻点弹出菜单里的「编辑」，可以进入布局一致的习惯编辑界面。

### 2. 联系人界面

联系人界面显示全部习惯中添加的所有联系人，可以一次性删除全部习惯里的某一个联系人。

通过轻点屏幕底部或侧边的「联系人」导航按钮，可以进入联系人界面。

处于联系人界面时，轻点屏幕底部或侧边的「联系人」导航按钮可以回到顶部。

### 3. 记录界面

记录界面显示每次打卡的习惯与时间。

通过轻点屏幕底部或侧边的「联系人」导航按钮，可以进入记录界面。

处于记录界面时，轻点屏幕底部或侧边的「记录」导航按钮可以回到顶部。
