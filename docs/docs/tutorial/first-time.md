# 初次上手 HabitPulse

HabitPulse 非常易于上手。即使是初次使用，也能飞快适应。

> [!IMPORTANT] 重要提示
> 本站教程仅适用于最新版本的 HabitPulse，如果你设备上的 HabitPulse 与教程里不一样，请更新 HabitPulse。

接下来我们来认识一下 HabitPulse 的各个页面。

## 习惯页面

习惯页面是应用启动时的默认页面，你添加的习惯以卡片列表样式显示在习惯页面。

通过轻点屏幕底部或侧边的「习惯」导航按钮，可以进入习惯页面。

处于习惯页面时，轻点屏幕底部或侧边的「习惯」导航按钮可以回到顶部。

### 习惯创建页面

这个页面同时用于新建习惯和编辑现有习惯。

通过轻点习惯页面右下角的「新建习惯」按钮，可以进入习惯创建页面。

通过长按任一现有习惯，再轻点弹出菜单里的「编辑」，可以进入布局一致的习惯编辑页面。

这个页面用于重排序习惯和批量删除习惯。

## 联系人页面

联系人页面显示全部习惯中添加的所有联系人，可以一次性删除全部习惯里的某一个联系人。

通过轻点屏幕底部或侧边的「联系人」导航按钮，可以进入联系人页面。

处于联系人页面时，轻点屏幕底部或侧边的「联系人」导航按钮可以回到顶部。

## 记录页面

记录页面显示每次打卡的习惯与时间。

通过轻点屏幕底部或侧边的「联系人」导航按钮，可以进入记录页面。

处于记录页面时，轻点屏幕底部或侧边的「记录」导航按钮可以回到顶部。

设置页面用于设置 HabitPulse。

处于习惯页面、联系人页面、记录页面时，轻点屏幕右上角的齿轮图标，可以进入设置页面。
