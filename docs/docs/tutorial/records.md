## 删除记录

打卡记录不能从记录界面删除。

如果你希望撤回某次习惯最近一次打卡，可以转到习惯页面，长按那个习惯卡片，再轻点弹出菜单里的「撤销打卡」。

如果你要多次撤回，需要重复这一操作。
