> [!IMPORTANT] 重要提示
> 本站教程仅适用于最新版本的 HabitPulse，如果你设备上的 HabitPulse 与教程里不一样，请更新 HabitPulse。

## 查看哪些习惯添加了这个联系人

在联系人页面，轻点列表里的联系人卡片，会弹出一个底部表单，里面显示使用此联系人作为监督人的全部习惯。

## 删除联系人

> [!CAUTION] 需要注意
> **删除联系人操作不可逆，确认删除前请三思！**

如果你需要从全部习惯中删除某个联系人，长按这个联系人，再轻点弹出菜单里的「从全部习惯删除」。

此时会弹出确认删除的对话框。轻点「删除」后，这个联系人从所有习惯中删除。
