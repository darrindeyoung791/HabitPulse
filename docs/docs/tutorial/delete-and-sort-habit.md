# 删除与排序习惯

> [!IMPORTANT] 重要提示
> 本站教程仅适用于最新版本的 HabitPulse，如果你设备上的 HabitPulse 与教程里不一样，请更新 HabitPulse。

## 删除一个习惯

> [!CAUTION] 需要注意
> **删除习惯操作不可逆，确认删除前请三思！**

长按你希望删除的习惯，再轻点弹出菜单里的「删除」。

此时会弹出确认删除的对话框。轻点「删除」后，这个习惯连同其打卡记录被删除。

## 删除多个习惯

> [!CAUTION] 需要注意
> **删除习惯操作不可逆，确认删除前请三思！**

HabitPulse 支持批量删除习惯。

长按任一现有习惯，再轻点弹出菜单里的「多选与排序」，可以进入习惯多选与排序界面。

## 重新排序习惯

最新编辑的习惯会处于首位。你也可以自己重新排序。

长按任一现有习惯，再轻点弹出菜单里的「多选与排序」，可以进入习惯多选与排序界面。

列表中每项习惯的右侧有拖拽柄，按住并上下拖动可以改变习惯的位置。

## 多选与排序界面常见问题

- **先前选中的习惯在拖动排序后会取消选中？**

    这是正常现象。HabitPulse 通过这个机制让保存重排序更简单。

- **页面右上角按钮不显示「保存」？**

    这是因为选中了习惯，页面右上角按钮变为「删除」。
    
    要显示「保存」，可以通过再次拖动习惯，或是轻点两次「全选」来取消选中。
