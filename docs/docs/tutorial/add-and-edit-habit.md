# 新建与编辑习惯

> [!IMPORTANT] 重要提示
> 本站教程仅适用于最新版本的 HabitPulse，如果你设备上的 HabitPulse 与教程里不一样，请更新 HabitPulse。

## 新建习惯

轻点屏幕右下角的「新建习惯」按钮，进入习惯创建界面。

填写完全部项目后，轻点右上角「保存」按钮，这个新建习惯就会保存到习惯列表第一个。

### 重复周期

HabitPulse 支持设置每日重复的习惯和每星期重复的习惯。
例如：
- 每日喝水
- 每周四给我￥50

HabitPulse 默认选择每日重复。要新建每星期重复的习惯，轻点「每周」，随后轻点选中至少一个需要重复的日期。

### 习惯名称

习惯名称是必填项。

### 提醒时间

HabitPulse 支持设置多个提醒时间，对应每天多次完成。

提醒时间是必填项。轻点「添加提醒时间」按钮，随后在弹出的时间选择器上选择一个时间添加为提醒时间。

已添加的时间，可以轻点「展开」，随后轻点其右侧的垃圾桶图标删除。

> [!TIP] 时间选择器怎么用？
> 要选中一个时刻，先轻点选择上午或是下午。
>
> 然后长按并拖动时钟上的旋转柄以选择小时，转到想要的小时后再松手。如果选错，先轻点小时数字，再次长按并拖动旋转柄。
> 
> 接下来，用同样的方式选择分钟。
>
> 或者，你可以点右下角的键盘图标切换到键盘输入。

> [!CAUTION] 需要注意
> **一般语境里的「中午 12 点」需要选择「下午」。因为「上午 12 点」实际上对应午夜 0 点。**

### 监督方式

HabitPulse 支持设置邮件通知和短信通知，用来让他人监督你。

监督方式是必填项。不监督时除外，其他情况需添加至少一个监督人的联系信息。

HabitPulse 默认选中不监督。

已添加的联系人，可以轻点「展开」，随后轻点其右侧的垃圾桶图标删除。

### 备注信息

备注不是必填项。你可以备注多行文本信息。

## 编辑习惯

在习惯界面，长按任一现有习惯，再轻点弹出菜单里的「编辑」，即可编辑。

编辑完全部项目后，轻点右上角「保存」按钮，这个新建习惯就会保存到习惯列表第一个。
