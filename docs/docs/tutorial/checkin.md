## 打卡与通知监督人

点击习惯界面卡片上的「✔️」按钮，即完成一次打卡。

轻点「完成」按钮或返回即可继续完成其他操作。

如果你打卡的这个习惯有添加监督人，完成打卡后会弹出一个底部表单，用来鼓励你完成打卡，并且询问你是否需要通知监督人。

轻点「通知 某习惯 的监督人」按钮，即可进入提醒界面。如果本次不想提醒，轻点「本次不通知」或者返回即可。

## 撤回打卡

如果你希望撤回某次习惯最近一次打卡，可以长按那个习惯卡片，再轻点弹出菜单里的「撤销打卡」。

如果你要多次撤回，需要重复这一操作。
